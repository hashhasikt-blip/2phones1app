## 📱 Kurulum ve Başlarken

### Sistem Gereksinimleri
- Android Studio Giraffe (2022.3.1) veya üzeri
- Android SDK 26 (Android 8.0) veya üzeri

- Kotlin 1.9.0+

1. Android Studio'yu açın
2. "Open an Existing Project" seçin
3. `/home/ke/2phones1app` klasörünü seçin
4. Wait for Gradle sync to complete

```bash

./gradlew installDebug
```

## 🔧 Ana Modüller

### 1. BluetoothManager
Bluetooth SPP profiyor üzerinden iki cihaz arasında iletişim.

**Kullanım:**
```kotlin
val btManager = BluetoothManager(context)
val devices = btManager.getAvailablePairedDevices()
btManager.connectToDevice(devices[0])

// Mesaj gönder
btManager.sendMessage(
    BluetoothMessage(
        type = "incoming_call",
        data = "+905551234567"
    )
)

// Mesaj dinle
lifecycleScope.launch {
    btManager.receivedMessages.collect { message ->
        // İşle
    }
}
```

### 2. CallMonitoringService
Telefon çağrılarını izler ve Bluetooth üzerinden yönlendirir.

**Başlatıldığında:**
- CallReceiver, PHONE_STATE_CHANGED olayında tetiklenir
- Service başlatılır ve çağrı bilgisi Bluetooth'a gönderilir
- Ses işlenmesini başlatır (gerekirse)

### 3. AudioRouting
Yerel ses kaydı ve oynatma işlemlerini yönetir.

```kotlin
val audioManager = getSystemService(AUDIO_SERVICE) as AudioManager
val audioRouting = AudioRouting(audioManager)

// Mikrofonu başlat
audioRouting.startRecording("/path/to/audio.3gp")

// Ses oynat
audioRouting.startPlayback(audioData)

// Temizle
audioRouting.cleanup()
```

## 📊 Bluetooth Protokolü

Mesajlar `TYPE|DATA` formatında iletilir:

| Tip | Örnek | Açıklama |
|-----|-------|----------|
| `incoming_call` | `incoming_call\|+905551234567` | Gelen çağrı |
| `call_connected` | `call_connected\|+905551234567` | Çağrı bağlandı |
| `call_ended` | `call_ended\|ended` | Çağrı sonlandırıldı |
| `contacts` | `contacts\|1,Ali,5551111\|2,Bob,5552222` | Rehber listesi |
| `audio_data` | `audio_data\|<base64>` | Ses verisi |
| `call_state` | `call_state\|added:+905551234567` | Durum değişimi |

## 🚀 Kullanım Akışı

### Senaryo 1: Çağrı Alma (Telefon A → Telefon B)

1. Telefon A'ya çağrı gelir
2. Sistem CallReceiver'ı tetikler
3. CallMonitoringService başlar
4. Çağrı numarası Bluetooth üzerinde Telefon B'ye gönderilir
5. Telefon B'de IncomingCallActivity gösterilir
6. Kullanıcı **Kabul** veya **Reddet** tıklar
7. **Kabul** durumunda:
   - AudioRouting ses işlemeyi başlatır
   - Telefon B'de ses işitilir ve konuşulur
   - Telefon A'da CallMonitoringService çağrı takip eder

### Senaryo 2: Rehber Görüntüleme

1. MainActivity'nin Bluetooth bağlı olması gerekir
2. ContactManager cihaz rehberini okur
3. "Refresh Contacts" basıldığında rehber Bluetooth üzerinde paylaşılır
4. Diğer cihaz rehberi TextView'de gösterir

### Senaryo 3: Arama Yapma (Telefon B)

1. Telefon B'deki MainActivity'de **"Refresh Contacts"** tıklanır
2. Rehber yüklenir
3. ContactManager contacts Flow'unu günceller
4. UI rehberi gösterir

## 🛠️ Geliştirme İpuçları

### Verbos Logging Etkinleştir
Her modül kendi TAG'ı kullanır:

```kotlin
Log.d("BluetoothManager", "Bağlanılıyor...")
Log.e("AudioRouting", "Hata oluştu", exception)
```

Logcat'te filtrele: `BluetoothManager` veya `CallMonitoringService`

### Bluetooth Bağlantısı Sorun Gidermesi

1. **Cihazlar Eşleştirilmiş mi?**
   - Ayarlar → Bluetooth → Eşleştirilmiş Cihazlar

2. **MAC Adresleri Alıyormu?**
   ```kotlin
   val devices = bluetoothManager.getAvailablePairedDevices()
   devices.forEach { device ->
       Log.d("TEST", "Device: ${device.name} - ${device.address}")
   }
   ```

3. **Socket Bağlanıyor mu?**
   ```kotlin
   Log.d("TEST", "Connected: ${bluetoothSocket?.isConnected}")
   ```

### Yeni Bluetooth Mesaj Türü Ekleme

1. **BluetoothMessage'de tür adlandırması yap**
   ```kotlin
   data class BluetoothMessage(
       val type: String,  // "incoming_call", "custom_type", vb.
       val data: String
   )
   ```

2. **CallMonitoringService'de işle**
   ```kotlin
   private fun handleBluetoothMessage(message: BluetoothMessage) {
       when (message.type) {
           "custom_type" -> {
               // İşle
           }
       }
   }
   ```

3. **Gönder**
   ```kotlin
   bluetoothManager.sendMessage(
       BluetoothMessage("custom_type", "verisi")
   )
   ```

## 🐛 Bilinen Sorunlar ve Çözümler

| Sorun | Neden | Çözüm |
|-------|-------|-------|
| Bluetooth bağlanmıyor | Cihazlar eşleştirilmedi | Bluetooth Ayarlarında manuel eşleştir |
| Ses işitilmiyor | Ses modu yanlış | AudioRouting.setAudioDevice() çağır |
| Çağrı yönlendirilmiyor | Manifest'te CallReceiver kayıtlı değil | AndroidManifest.xml kontrol et |
| Runtime Permission hatası | İzinler verilmedi | MainActivity requestPermissions() çağır |
